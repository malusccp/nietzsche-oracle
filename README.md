# 📜 Nietzsche Oracle

A **Full-Stack project** built to practice and demonstrate the integration between a Database, a REST API, and a themed Frontend.

## 🌑 The Project
The goal was to create a simple yet creative interface to display thoughts from Friedrich Nietzsche and consolidate knowledge in Database management, Backend architecture, and Custom UI/UX development.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, and Vanilla JavaScript (Fetch API).
* **Backend:** Node.js and Express (REST API).
* **Database:** MongoDB and Mongoose (Storing 1.4k+ quotes).

## ⚙️ Main Tasks Performed
* **Database Seeding:** Created a Node.js script to clean and import 1,400+ quotes into MongoDB.
* **API Development:** Built a backend to serve random quotes to the frontend.
* **Custom UI:** Designed a responsive layout with CSS and SVG masks.
